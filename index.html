Sobre

O objetivo desse aplicativo é facilitar o uso do sistema de transporte público de Fortaleza. Para isso, o App consome dados da API (Application Programming Interface) da ETUFor (Empresa de Transporte Urbano de Fortaleza). Por tanto, Todas as informações mostradas neste App são fornecidas pela ETUFor.

O que você pode fazer com esse App?

1 - Ver a lista de todas as linhas de ônibus de Fortaleza 2 - Encontrar uma linha específica 3 - Ver o intinerário de uma determinada linha 4 - Ver os horários de uma determinada linha em seus postos de controle. Um posto de controle pode ser um terminal ou qualquer outro logradouro (rua, avenida etc...) 5 - Ver todos os logradouros (terminais, ruas, avenidas etc...) servidos por pelo menos uma linha. 6 - Encontrar um logradouro (terminais, ruas, avenidas etc...) específico. Ou seja: você pode pesquisar por um logradouro qualquer de Fortaleza desde que, neste logradouro, passe pelo menos uma linha de ônibus 7 - Vêr todas as linhas de ônibus que passam em um determinado logradouro (terminais, ruas, avenidas etc...)

Política de privacidade

1 - Esse aplicativo não precisa de qualquer permissão especial 2 - Nenhuma informação do usuário é enviada para qualquer dispositivo

Observações


Programação não encontrada!

Ao pesquisar sobre horários de linhas nos postos de controle, você poderá se deparar com a mensagem "Programação não encontrada!". Isso significa que a API (Application Programming Interface) da ETUFor não está fornecendo os horários para essa determinada linha de ônibus. Tente novamente outro dia.

Sugestão de ônibus

Para receber sugestão de ônibus o usuário escolhe dois logradouros e o App pesquisa por linhas de ônibus que passem pelos dois logradouros fornecidos. Ao encontrar sugestões, o usuário poderá selecionar qualquer uma delas e vêr o itinerário completo da linha.
